# My Vue Sheet 表单

> 使用 WeUI 与 Mint UI  

## 项目预览

[https://www.rejiejay.cn/my-vue-sheet/](https://www.rejiejay.cn/my-vue-sheet/)

## 项目搭建

``` bash
# 初始化项目
npm install

# 运行开发环境, 服务端端口热加载 localhost:8080
npm run dev

# 运行生产环境
npm run build

```

更多的请查看 [Vue](https://cn.vuejs.org/)、 [WeUI](https://weui.io/)、 [Mint UI](http://mint-ui.github.io/#!/zh-cn)
